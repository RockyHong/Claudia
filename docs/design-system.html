11 Rules

Spacing — Use --space-{n} tokens only. No raw pixel values in component styles.

Radius — Five stops: 4, 6, 8, 12, 16. Nothing else.

Hierarchy — Differentiate by color weight first, font-size second. Don't inflate size to create emphasis.

Buttons — Three sizes (sm/md/lg), three variants (primary/ghost/danger). Pick one of each. No custom padding.

Actions — Right-aligned. Primary rightmost. Gap: --space-2.

Colors — Use semantic vars. Never hardcode hex in components. var(--red) not #ef4444.

Units — rem for font sizes (via tokens), px for spacing/radius (via tokens). Never mix raw units.

Modals — Use Modal.svelte. Sub-modals get back arrow. Confirms are compact. Gates are full-screen centered.

Section labels — One pattern: xs, 600, uppercase, 0.08em, --text-faint. No variations.
