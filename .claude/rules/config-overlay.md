# Config & Overlay Authoring

You are editing runtime wiring config — an overlay surface.

- **Default to upstream canonical wiring as shipped.** Empirically prove canonical fails before authoring an overlay; date-stamp any overlay you keep as decay debt → read `.claude/guidelines/work-discipline/trust-upstream-defaults.md`
- **Place ambient-behavior config at the layer every target runtime loads.** Name the runtimes, identify their loading layers, pick the lowest-common one; document parity exceptions explicitly → read `.claude/guidelines/work-discipline/runtime-parity.md`
